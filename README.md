# typescript - dreamcording

## typescript 설치

1. typescript 글로벌 설치

```
npm install -g typescript
```

2. tsc 명령어를 이용하여 .js 파일로 컴파일

```
tsc main.ts
```

3. 터미널에서 컴파일과 실행까지 한번에 처리 ts-node 설치

```
npm install -g ts-node
ts-node script.ts
```

4. tsc watch 모드 -w

```
tsc main.ts -w
```

> 프로그래밍 3가지 순서
> input(db) => operation => output
